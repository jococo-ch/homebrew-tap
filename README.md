# homebrew-tap

Homebrew tap for [jococo-ch](https://github.com/jococo-ch) tools.

## Usage

    brew tap jococo-ch/tap
    brew install lola-sumup

## Tools

| Tool | Description |
|---|---|
| [lola-sumup](https://github.com/jococo-ch/lola-sumup) | CLI to create LoLa-specific exports from SumUp reports |
